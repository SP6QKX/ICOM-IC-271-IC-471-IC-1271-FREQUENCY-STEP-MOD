# ICOM-IC-271 IC-471 IC-1271 FREQUENCY STEP MOD 

             SETTINGS STEP FM FREQUENCY FOR 
             ICOM IC-271 / IC-471 / IC-1271  

Select-matrix are on the LOGIC Unit 
From the IC6 PIN11 are comon connection matrix 
diodes Anode( 1N4148 or similar ) to IC12 PIN 3,4,7,8 Cathode 

Step           Diodes instaled
==========================================================
10.Hz          Without any diodes  
100.Hz         D20
1.Khz          D19
5.Khz          D19 and D20
10.Khz         D21
12.5Khz        D19 and D21
25.Khz         D21a - ( sometimes marked as D34 ) 
1.Mhz          D21 and D21a -( sometimes marked as D34 ) 
D21a / sometimes marked D34 only used in IC-471 / IC-1271
